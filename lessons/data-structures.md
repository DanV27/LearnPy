Python's built-in `list`, `dict`, `set`, and `tuple` cover most everyday needs — those live under [Basics](/lesson/basics). This section is for the next layer up: stacks, queues, heaps, linked lists, trees, tries, graphs, and the underused gems in the `collections` module.

For each one you'll find a short explanation, a working code example, and notes on when (and when not) to use it. Click anything in the grid below to get started.
