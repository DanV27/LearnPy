Dan's Code Generator

An AI-powered code generation tool that writes Python code from plain English descriptions — then writes the tests, runs them, and fixes itself if anything breaks.

Status: Fully Functional V: 1.1
bolt

What It Does

  1. 1

    You describe what you want

    Write a plain English spec — for example, "a function that adds two numbers."

  2. 2

    Claude generates code

    The AI writes clean, documented Python with proper type hints.

  3. 3

    Tests are auto-generated

    A matching pytest suite is generated alongside the code.

  4. 4

    Code is validated

    Syntax is checked and the test suite is executed in a sandbox.

  5. 5

    Problems are fixed automatically

    If tests fail, the code is regenerated to make them pass.

  6. 6

    You get the results

    Clean code, test results, and complexity metrics — all in one shot.
