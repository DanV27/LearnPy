LearnPy

Learn Python by building the things you keep reading about. Type a topic — a data structure, an algorithm, a standard-library trick — and LearnPy implements it, tests it, and walks you through how it actually works.

Think of it as a personal Python tutor focused on one language done well — an alternative to the ad-heavy, copy-paste-friendly reference sites. Every lesson is a working, tested implementation you can run.

Status: In Development Python 3.9+
school

How a Lesson Works

  1. 1

    Browse the sidebar (or search)

    Open the Topics drawer to see the curated catalog, or type a question into the search bar on the home page. The catalog covers Python fundamentals, data structures, algorithms, and standard-library tools.

  2. 2

    Read the hand-authored lesson

    Every lesson page is a short, focused write-up with at least one working code example. No ads, no SEO bloat, no "scroll past 40 paragraphs to find the snippet you want".

  3. 3

    Try the challenge

    Every lesson has a coding challenge that runs in your browser. Write Python, click Run Tests, and see pass/fail per test instantly. No setup, no server, no risk — your code never leaves your machine.

  4. 4

    Keep exploring

    Every page ends with related-topic links and four follow-up question buttons ("Explain more", "Why do I need this?", and so on) that summon an AI tutor for a deeper take when you want it.

Hands-On Challenges

Reading code isn't enough — you have to write some. Every lesson ends with a small coding challenge that you solve right on the page.

  • A prompt explains what to build, with starter code pre-filled in the editor.
  • Click Run Tests and your code is executed in the browser via Pyodide — real Python, no server, no upload.
  • Each test reports pass or fail with a clear error message when something's off.
  • Hit a green ✓ CHALLENGE COMPLETE badge when every test passes.
  • Reset rolls back to the starter; Cmd/Ctrl + Enter runs the tests.

Can't find your topic? Just search it.

The catalog can't cover everything. If you don't see your topic in the sidebar, head to the home page, type a question into the search bar, and the AI tutor will write you a custom lesson — short, focused, and linked to whichever canonical lessons relate.

Examples that work great: "How do I read CSV files in Python?", "Implement Dijkstra's algorithm", "What's the difference between == and is?".
