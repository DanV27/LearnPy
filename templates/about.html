LearnPy

Learn Python by building the things you keep reading about. Type a topic — a data structure, an algorithm, a standard-library trick — and LearnPy implements it, tests it, and walks you through how it actually works.

Think of it as a personal Python tutor focused on one language done well — an alternative to the ad-heavy, copy-paste-friendly reference sites. Every lesson is a working, tested implementation you can run.

Status: In Development Python 3.9+
school

How a Lesson Works

  1. 1

    You pick a topic

    Anything specific in Python — "implement a binary search tree", "how do I parse JSON?", "build a hash map from scratch".

  2. 2

    LearnPy writes the implementation

    Clean, idiomatic Python with type hints and docstrings — the way it should be written, not the shortest possible version.

  3. 3

    Tests are generated alongside

    A matching pytest suite covers normal, edge, and error cases — so you can see what the code is supposed to do.

  4. 4

    Everything runs in a sandbox

    Syntax is checked and the tests are executed before you see the lesson. If they fail, LearnPy revises the code until they pass.

  5. 5

    You get a verified lesson

    Working code, the tests that prove it, and complexity stats so you know what you're looking at.
